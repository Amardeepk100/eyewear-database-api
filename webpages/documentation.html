Introduction

What is this?

Eyewear Database API is a REST API based on my previous project Elite Eyewear. The Eyewear Database API contains data of numerous eyewear products from the previous project.

Try it now!

http://localhost:5000/ http://localhost:5000/
 Get

Root URL

This is the root url. All requests are GET requests and are sent over HTTPS.

http://localhost:5000/

Limiting and Skipping

All endpoints support the limit and skip parameters. You can use this parameters together or separately. The default value of limit is 5 if the value is not provided by the user.

http://localhost:5000/products?limit=1&skip=5

Filtering Endpoints

All endpoints support filtering on any available key within their schema.

http://localhost:5000/products?product_type=Eyeglasses&
frame_shape=Cat Eye

Filtering By Price

Products can be filter by price using minPrice and maxPrice parameters. This parameters can also be used with other endpoints.

http://localhost:5000/products?product_type=Eyeglasses&
minPrice=1000&maxPrice=2000

Search

Products can be searched across multiple fields using any given value with the help of search endpoint.

http://localhost:5000/products/
search?q=geometric
